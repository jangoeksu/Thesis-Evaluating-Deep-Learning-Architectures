# Data

This project uses two English-language news classification datasets:

1. **AG News**
   - Files used locally:
     - `data/raw/AG_train.csv`
     - `data/raw/AG_test.csv`

2. **Kaggle News Category Dataset**
   - File used locally:
     - `data/raw/Kaggle_News.json`

The raw datasets are not committed to this repository. They must be downloaded separately from their original Kaggle sources and placed in `data/raw/` before running the preprocessing pipeline.

## Folder structure

- `raw/` contains the original downloaded datasets without manual modification.
- `processed/` contains cleaned, label-mapped, and split datasets generated by the preprocessing code.

## Expected processed outputs

After running the preprocessing pipeline, the `processed/` folder will contain reproducible train, validation, and test files for both datasets, together with supporting preprocessing summaries where applicable.

These processed files are generated automatically and are not required to be added manually.
